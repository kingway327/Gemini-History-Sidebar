# Gemini History Sidebar (Gemini 历史记录侧边栏)

一个仿照deep seek历史边栏做的，为 Google Gemini 打造的增强型扩展，提供丝滑的历史记录管理与页面内导航体验。


## ✨ 核心功能

- **智能历史侧边栏**：在弹窗中快速查看、搜索并跳转到您的 Gemini 对话历史。
- **动态页面导航**：自动提取当前对话中的所有用户提问，并在页面右侧生成星星导航点。
- **滚动感应 (Scroll Sync)**：页面滚动时，右侧星星导航自动切换高亮，侧边栏列表同步跟随。
- **玻璃拟态设计**：采用现代 UI 设计，支持自动隐藏/显示，并提供“固定”功能。
- **隐私保护**：所有数据均在本地处理，自动爬取当前登录用户信息，无后端传输。
- **快捷键支持**：`Alt + H` (Windows) 或 `Command + Shift + H` (Mac) 快速唤起。

## 🛡️ 安全性与可靠性说明

### 1. 隐私安全 (Privacy First)
- **零数据上传**：本插件没有后端服务器。所有的对话历史提取、用户信息展示均在您的浏览器本地完成。
- **最小权限原则**：仅申请了必要的 `tabs` 和 `storage` 权限，且仅在 `gemini.google.com` 域名下运行。
- **无敏感信息访问**：插件仅读取页面上已有的 HTML 文本（对话标题），无法访问您的 Google 密码、支付信息或其他私密数据。

### 2. 可靠性保障 (Reliability)
- **防抖处理**：数据抓取逻辑带有智能防抖和频率限制，不会影响 Gemini 网页本身的性能。
- **降级方案**：当页面结构发生变化导致抓取失败时，插件会保持静默或显示友好提示，不会导致页面崩溃。
- **资源优化**：图标与 UI 经过压缩处理，占用极低的内存和 CPU 资源。

## � 快速开始

### 开发环境配置
1. 克隆或下载本仓库。
2. 安装依赖：
   ```bash
   npm install
   ```
3. 运行开发服务器：
   ```bash
   npm run dev
   ```

### 构建与安装
1. 执行构建命令：
   ```bash
   npm run build
   ```
2. 打开 Chrome 浏览器，进入 `chrome://extensions/`。
3. 开启右上角的 **“开发者模式”**。
4. 点击 **“加载已解压的扩展程序”**，选择项目根目录下的 `dist` 文件夹。

## �️ 技术栈

- **框架**: React 18 + TypeScript
- **构建工具**: Vite + @crxjs/vite-plugin
- **样式**: Tailwind CSS
- **交互**: 原生 DOM 监听 + Intersection Observer API

## � 项目结构

- `src/content.ts`: 核心逻辑，负责页面数据提取、UI 注入与滚动监听。
- `src/App.tsx`: 插件弹窗的主界面逻辑。
- `public/`: 存放静态资源（如插件图标）。
- `dist/`: 构建后的最终产物。
