# Gemini History Sidebar (Gemini 历史记录侧边栏)

一个仿照deep seek历史边栏做的，为 Google Gemini 打造的增强型扩展，提供丝滑的历史记录管理与页面内导航体验。
下载地址：https://github.com/kingway327/Gemini-History-Sidebar/releases/tag/v1.0.0

## ✨ 核心功能

- **智能历史侧边栏**：在弹窗中快速查看、搜索并跳转到您的 Gemini 对话历史。
- **动态页面导航**：自动提取当前对话中的所有用户提问，并在页面右侧生成星星导航点。
- **滚动感应 (Scroll Sync)**：页面滚动时，右侧星星导航自动切换高亮，侧边栏列表同步跟随。
- **玻璃拟态设计**：采用现代 UI 设计，支持自动隐藏/显示，并提供“固定”功能。
- **隐私保护**：所有数据均在本地处理，自动爬取当前登录用户信息，无后端传输。
- **快捷键支持**：`Alt + H` (Windows) 或 `Command + Shift + H` (Mac) 快速唤起。

## � 快速开始

### 构建与安装
1. 下载压缩包，并解压。
2. 打开 Chrome 浏览器，进入 `chrome://extensions/`，edge是`edge://extensions/`。
3. 开启 **“开发者模式”**。
4. 点击 **“加载已解压的扩展程序”**，选择刚刚解压到的文件夹。
